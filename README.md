# Native runtime
* Install graalvm
* Set java home to graalvm
* Install VS 2019 build tools
* Setup environment with "C:\Program Files (x86)\Microsoft Visual Studio\2019\BuildTools\VC\Auxiliary\Build\vcvars64.bat"
* Run mvn -Pnative native:compile
